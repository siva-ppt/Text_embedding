# AI Benchmarks

This project benchmarks text embedding models using MTEB and Sentence Transformers.

## Structure

- `shared/`: Shared utilities and benchmarking code.
- `articles/01-text-embeddings/`: Benchmarking for text embeddings.
- `docs/`: Documentation and guides.

## Setup

1. Install requirements: `pip install -r requirements.txt`
2. Run benchmarks: `python articles/01-text-embeddings/run_benchmarks.py`
